# 段落和行

- text-indent
  - 定义首行文本内容之前的缩进量
  - 缩进两个字符写作
    - text-indent: 2em
    - em表示字符宽度

- line-height
  - 用于定义行高
  - ![行高](./%E8%A1%8C%E9%AB%98.PNG)
  - 属性的单位可以是以px为单位的数值
    - line-height: 30px
  - 也可以是没有单位的数值，表示字号的倍数
    - line-height: 1.5
  - 也可以是百分数，表示字号的倍数
    - line-height: 150%

  - ![单行文本垂直居中](./%E5%8D%95%E8%A1%8C%E6%96%87%E6%9C%AC%E5%9E%82%E7%9B%B4%E5%B1%85%E4%B8%AD.PNG)

- font合写属性
  -可以用来作为font-style，font-weight，font-size, line-height和font-familt属性的合写
