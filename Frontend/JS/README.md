## 前端三层

||语言|功能|
|-|-|-|
结构层|HTML|搭建结构、放置部件、描述语义
样式层|CSS|美化页面、实现布局

## ECMAScript是JavaScript的标准

1997年，欧洲计算机制造商协会（ECMA）设置了JavaScript的标准，命名为ECMAScript
